# Event Registration App

## Description
A simple event registration web app that allows users to enter their name and email.

## Features
- Registration form
- Basic input fields (name, email)
- Simple form validation using JavaScript
- Clean and minimal UI

## Technologies Used
- HTML
- CSS
- JavaScript
- Git & GitHub

## How to Run
1. Open the project folder
2. Open `index.html` in a browser
3. Fill in the form and click register
